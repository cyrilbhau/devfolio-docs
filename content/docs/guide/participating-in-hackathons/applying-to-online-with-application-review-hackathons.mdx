Hackathons set up in `Online With Application Review` mode are strictly **team hackathons**.

You can apply to such hackathons individually and get accepted, but in order to submit the project, it is mandatory to **join or create a team**.

Once accepted, you can RSVP within the time period allotted so that the organizer can check you in!

<Callout type="warning">
All the team members of the hackathon need to RSVP and be checked-in. Without that, the team won't be able to make a submission.
</Callout>
